## Umiejętności

**SIEM / Blue Team**
- Wazuh – instalacja agentów, analiza alertów, threat hunting, vulnerability detection
- Sysmon – konfiguracja i integracja z SIEM
- File Integrity Monitoring, Security Configuration Assessment (CIS Benchmark)

**Pentesting / Red Team**
- Metasploit Framework – exploitacja, MSFvenom, reverse shell
- Burp Suite Community Edition – proxy, web application testing
- Narzędzia rekon: nmap, Nikto, Hydra
- Eksploitacja podatności CVE (EternalBlue MS17-010, CVE-2024-21413, CVE-2023-30258)
- Web: SQL injection, IDOR, SSRF, file upload bypass, XSS

**Systemy i sieci**
- Linux (Kali, Ubuntu) – administracja, iptables, UFW, systemd
- Windows – PowerShell, usługi systemowe
- Sieci: VPN (OpenVPN/TryHackMe), nmap scanning, podstawy protokołów
